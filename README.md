# YT Sub Fetcher

Chrome Extension that catches YouTube's internal subtitle requests and allows you to download them instantly in various formats including ytt. 

Supported download formats: `srv3` (also known as YTT), `json3`, `srt`, `vtt`, `srv1`, `srv2`, and `ttml`.

---

## How to Install (Easiest Way)

Since this extension is not hosted on the Chrome Web Store, you can install it manually in **Developer Mode**.

### Step 1: Download the Extension
1. Click the green **Code** button at the top right of this GitHub page.
2. Click **Download ZIP**.
3. Extract the downloaded `.zip` file to a permanent folder on your computer (e.g., your Documents folder). *Do not delete this folder after installation, as Chrome loads the extension directly from it.*

### Step 2: Enable Developer Mode in Chrome
1. Open Google Chrome.
2. In the URL bar, type `chrome://extensions/` and press **Enter**.
3. In the top-right corner of the Extensions page, toggle the **Developer mode** switch to **ON**.

### Step 3: Load the Extension
1. Click the **Load unpacked** button in the top-left corner.
2. Select the extracted folder containing the extension files (the folder that has `manifest.json` inside it).
3. The extension is now installed! You will see it appear in your extensions list.

---

## How to Use

1. Go to any YouTube video.
2. **Important:** If subtitles are turned off by default, click the **CC** icon on the YouTube player for a split second to force Chrome to request the subtitle track.
3. Click the **Puzzle piece icon** (Extensions menu) in your Chrome toolbar and click **YT Sub Fetcher** (you can also pin it for quicker access).
4. Select your desired format from the popup list to instantly download the subtitle file.
